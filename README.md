# PubMed Biomedical RAG Assistant

## Objective

Build a Retrieval-Augmented Generation (RAG) system for biomedical literature.

## Pipeline

PubMed
→ Embeddings
→ ChromaDB
→ Local LLM (Ollama)
→ Literature-grounded answer

## Models Benchmarked

- Qwen3.5
- Qwen3:4B
- Bio-Mistral
- DeepSeek-R1
- Phi-4
- Gemma3
- Llama3
- OpenBioLLM

## Results

Qwen3.5 achieved the highest answer quality.
Qwen3:4B provided the best quality-speed tradeoff.
